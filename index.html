PIPE

One platform to query, join, analyze and visualize, any data source

Query any database

Pipe comes with built-in support for the following databases:

  • SQL: (MySQL, PostreSQL, MSSQL)
  • MongoDB
  • RethinkDB
  • ElasticSearch
  • Redis
  • SharePoint
  • Any Web API (using cURL) or Stream (using Sockets)

Pipe queries can be composed with each other for creating larger computations like joining MongoDB with a SQL database.

Transform the result

Some analysis are only possible or easier and faster done on the result of the query.

For example if you want to find how many standard deviation each of your data points are away from their mean, without Pipe you had to first execute a query, copy its result into Excel or R and then do your analysis.

Use Pipe's innovative Transformation pipeline to manipulate the result of your query and perform statistical analysis directly on the result without having to copy and paste into another software. Use Pipe's built-in utilities or import your favorite JavaScript library.

Visualize your reports

Plottables are ready-to-use, standardized data visualization components. Plottables are composable and interactive.

Pipe comes with many built-in Plottables (check them out here).

Use Pipe's streaming features to share real-time visualizations.

You can also import 3rd party charting libraries or create your own custom visualizations or reusable Plottables in JavaScript.

Share storyboards

Pipe is a Gist for database queries and data analysis with collaboration, versioning, forking and access control.

You can create fast ad-hoc REST APIs in Pipe and share it with your customers and colleagues.

Create your queries in Pipe to share up-to-date reports with live visualizations instead of static Excel sheets.

Using Pipe's Storyboards you can create a full fledged dashboard and share it with your team in minutes.
