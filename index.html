PIPE

One platform to query, join, analyze and visualize, any data source

Query any database

Pipe comes with built-in support for the following databases:

  • SQL: (MySQL, PostreSQL, MSSQL)
  • MongoDB
  • RethinkDB
  • ElasticSearch
  • Redis
  • SharePoint
  • Any Web API (using cURL) or Stream (using Sockets)

Transform the result

Some analysis are only possible or easier and faster done on the result of the query at client-side.

Use Pipe's innovative Transformation block to manipulate the result of your query by built-in utilities and statistical functions. (or import your favorite JavaScript library)

Visualize your reports

Plottables are ready-to-use, standardized data visualization components. Pipe comes with many built-in Plottables (check them out here).

You can also create your own custom visualization or reusable Plottables in JavaScript.

Share storyboards

Pipe is a Gist for database queries and data analysis with collaboration, versioning, forking and access control.

Using Pipe's Storyboards you can create a full fledged dashboard and share it with your team in minutes.
